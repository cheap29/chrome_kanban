# Chrome Web Store Assets

Chrome Web Store掲載用の画像をこのディレクトリで管理します。

## Directory

- `screenshots/`: ストア掲載スクリーンショット
- `promo/`: プロモーション画像
- `../icons/icon128.png`: Chrome Web Store用アイコン

## Sizes

- `../icons/icon128.png`: 128x128 px
- `screenshots/screenshot-*.png`: 1280x800 px、最低1枚、最大5枚
- `promo/small-promo-440x280.png`: 440x280 px
- `promo/marquee-promo-1400x560.png`: 1400x560 px

## Screenshot Plan

1. `screenshot-01-main-board.png`: 今ここボード全体
2. `screenshot-02-drag-drop.png`: ドラッグ＆ドロップでレーン移動
3. `screenshot-03-complete-toast.png`: 完了時の「おめでと！」トースト
4. `screenshot-04-edit-task.png`: タスクの編集
5. `screenshot-05-local-save.png`: ローカル保存・外部送信なしの説明画面

## Notes

- ストア掲載画像は、ぼやけ・歪み・過度な装飾を避ける
- 実際の機能と一致する画面を使う
- ブランドの見た目をアイコン、スクリーンショット、プロモーション画像で統一する
